## 文件结构：
```
    ├── batch_predict.py: 批量图像预测脚本
    ├── class_indices.json：类别索引
    ├── handle.py：图像处理脚本
    ├── load_weights.py：加载权重脚本
    ├── main.py：主程序
    ├── model.py: ResNet模型搭建
    ├── my_dataset.py：数据集处理脚本
    ├── predict.py: 单张图像预测脚本
    ├── train.py: 训练脚本
    ├── train_eval_util.py: 与train_save相关工具
    ├── train_save.py: 保存模型到tensorboard
    ├── utils.py: 与train_save相关工具
    └── utils.py：工具函数
```
